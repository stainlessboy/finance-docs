Spendly ("Приложение") — мобильное приложение для учёта личных финансов. Настоящая Политика описывает, какие данные мы собираем, зачем и как их защищаем.

3. Чего мы НЕ делаем

  • Не продаём и не передаём ваши данные третьим лицам
  • Не показываем рекламу
  • Не используем данные для профилирования или аналитики сторонних сервисов
  • Не собираем контакты, фото, геолокацию или данные других приложений

4. Хранение и защита данных

Все данные хранятся на защищённом сервере с использованием шифрования соединения (HTTPS/TLS). Пароли хранятся в виде bcrypt-хеша и никогда не передаются в открытом виде.

Токены авторизации (JWT) имеют ограниченный срок действия. Сессии привязаны к конкретному устройству.

5. Сторонние сервисы

  • Firebase Cloud Messaging (FCM) — для доставки push-уведомлений. Политика Firebase
  • CBU API / open.er-api.com — для получения курсов валют. Ваши персональные данные не передаются этим сервисам.

6. Ваши права

Вы имеете право:

  • Просматривать все свои данные в приложении
  • Редактировать или удалять транзакции, счета, категории, цели и долги
  • Запросить полное удаление аккаунта и всех связанных данных
  • Отключить push-уведомления в настройках устройства

7. Удаление данных

Для полного удаления аккаунта и всех данных напишите нам на email ниже. Данные будут удалены в течение 30 дней.

8. Изменения политики

Мы можем обновлять данную Политику. Актуальная версия всегда доступна на этой странице. При существенных изменениях мы уведомим пользователей через приложение.

All data is stored on a secure server using encrypted connections (HTTPS/TLS). Passwords are stored as bcrypt hashes and are never transmitted in plain text.

  • Request complete deletion of your account and all associated data
